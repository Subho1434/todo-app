# cli-todo-app
1️⃣ cli-todo-app (Basic Version)

A simple Command-Line Interface (CLI) To-Do List application built in Python.  
This project demonstrates the fundamentals of CRUD operations, input handling, and basic Python programming.

## 🛠 Features

- Add tasks
- View all tasks
- Update tasks
- Delete tasks
- Simple CLI menu for interaction

## 💡 Skills Demonstrated

- Python basics (variables, loops, conditionals)
- CRUD operations
- Input/output handling
- List management and indexing

## 🚀 How to Run

1. Clone the repository:

git clone https://github.com/Subho1434/cli-todo-app.git

Navigate to the folder:

cd cli-todo-app

Run the application:

python todo.py

📝 Usage

Select an option from the menu (Add, View, Update, Delete, Exit)
Follow the prompts to manage your tasks

🎯 Goal

This project serves as a beginner-friendly demonstration of coding logic and program structure.
