# 📝 To-Do App (Python CLI)

A simple and beginner-friendly Python-based Command-Line To-Do application.

This repository contains different versions of the project, showcasing improvements and added features over time.

---

```
todo-app/

│   ├── tasks.txt
│   └── README.md      # Detailed documentation
│
├── LICENSE
└── .gitignore
```

## 🚀 Latest Version

👉 **v2 - CLI To-Do App**

Features:

* Add, update, delete tasks
* Persistent storage using a file
* Interactive CLI menu
* Colorful output using `colorama`

📖 For full documentation and usage instructions, see:
➡️ `v2/README.md`

## ⚡ Quick Start

```bash
git clone https://github.com/Subho1434/todo-app.git
cd todo-app/v2
python cli-todo-app-v2.py
```
